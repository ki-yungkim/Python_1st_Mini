## 파이썬 웹 미니 프로젝트
# 기간 : 210907~210909

# 조 구성 : 김기영, 이택근, 이희원 
# 조장 : 김기영 

# 주제 : 온라인 서점 시스템 구축 

# 기능 구성
- 회원가입, 내 정보 보기, 내가 구입한 책 목록(대여한 목록 포함)
- 관리자 모드 : 책 정보 추가
- 책 목록 보기
- 종이 책 구입 : 구입 시 재고 감소시키기
- E북 구입, 대여 : 대여 시 유효기간 정보 추가 
- 서평 작성
    - 추가 구상 중인 기능
        - 별 갯수로 추천도 표시 시킬 수 있는 기능
        - 좋아요 기능 

# DB 구성 
- 회원 테이블 : 아이디, 비밀번호, 이름, 이메일, 연락처 
- 관리자 테이블 : 아이디, 비밀번호, 이름

- 종이책 정보 테이블 : 분류, 제목, 출판사, 가격, 수량 
- E북 정보 테이블 : 분류, 제목, 출판사, 가격, 수량
- 서평 정보 테이블 : 작성자, 내용, 작성날짜, 점수(5점 만점)

- 구입 정보 테이블 : 구매자, 책 이름, 구입/대여 여부, 유효기간
